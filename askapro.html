Debugging

Main Page Debugging

When you build multiple thousand block builds it is guarenteed that you will sometimes break it, some of these are common mistakes and some are random oversights that are hard to find.

Frequent mistakes
  1. Massless must be applied to almost the entire build
  2. If your build is completely massless it will freeze or desync, requiring some mass to resync
  3. If your build does not have the correct staging then massless will not spread, use locked blade motors and springs for the core.
  4. Massless does not apply to motors that aren't hyper, if a motor is too far away from the core (horizontally not up or down) it will break your build.

Before you ask
