Massless Info

Massless modules work when a few requirements are met:

  • The motor2 has 100 torque or more
  • The object cannot be the root part of its assembly
  • The object has a rotational velocity magnitude of 1 or more

Desync

Due to a bug, massless can desynchronize assemblies. This happens because massless creates a local variable that stores all blocks, but never clears it.

Overcome this by intentionally manipulating root parts so your build contains the root part. Having mass within your build helps with this.

Staging

Massless does not spread to other assemblies, so you must temporarily weld them together with staging blocks (springs, disconnectors, or TNT).
